# Лабораторна робота №8 — Пошук телефонних номерів у тексті за допомогою регулярних виразів

## Опис проєкту

Програма читає текстовий файл `input.txt`, знаходить у ньому телефонні номери за форматом:

- XX-XX-XX
- XXX-XX-XX
- (XXX)-XX-XX
- +38(0XX)-XXX-XX-XX

Усі знайдені номери замінюються на маркер `$PHONE$`, після чого результат записується у `output.txt`.

---

## Структура проєкту
```
Lab8/
├── phone.l (файл з правилами Flex)
├── input.txt (вхідні дані)
└── README.md
```

---

## Як компілювати та запускати на **Linux / macOS**

1. Перейди в директорію проєкту
2. Скомпілюй програму
3. Запусти виконуваний файл

```bash
cd шлях/до/папки/Lab8
gcc main.c -o lab8
./lab8
```

> `lex.yy.c` та `output.txt` створюються після запуску.

---

## Інструкція запуску на Linux / macOS

1. Встанови Flex (якщо ще не встановлено)

macOS (через Homebrew):
```
brew install flex
```

Linux (Ubuntu / Debian):
```
sudo apt install flex build-essential
```

2. Перейди у директорію з проєктом:
```
cd шлях/до/Lab8
```

4. Згенеруй код Flex:
```
flex phone.l
```

6. Скомпілюй створений файл:
```
gcc lex.yy.c -o phone
```

8. Запусти програму:
```
./phone < input.txt > output.txt
```

## Формати телефонних номерів, які розпізнає програма

| Приклад            | Пояснення                   |
|-------------------|-----------------------------|
| 12-34-56          | стандартний формат          |
| 067-88-99         | короткий формат             |
| (095)-12-34       | формат з кодом у дужках     |
| +38(067)-123-45-67| міжнародний формат          |
