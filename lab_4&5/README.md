# Практична робота №4-5

## Варіант 8

### Завдання 4

Розроблено регулярний вираз для валідації рядків, що містять 10 полів (рік, температура, опади, код метеостанції тощо) з різними роздільниками (`:`, `;`, `?`).

### Завдання 5

Написано скрипт на **Python**, що виконує наступні операції:
1.  Читає вхідний файл `input.txt`.
2.  Валідує кожен рядок за допомогою регулярного виразу (Завдання 4).
3.  **Фільтрація:** Створює `output.txt`, що містить *лише* ті записи, де середня денна температура є невід'ємною (`>= 0`).
4.  **Статистика:** Створює `aux_stats.txt` з підрахунком загальної кількості валідних записів та кількості записів, де сила вітру перевищувала 15.

## Реалізація

Рішення реалізовано мовою **Python 3** з використанням вбудованої бібліотеки `re`.

* Для валідації рядка та захоплення полів використовується функція `re.match()`.
* Для перетворення рядка та заміни роздільників на `;` у вихідному файлі використовується виключно `re.sub()`, як того вимагають умови завдання.

## Запуск

1.  Клонувати репозиторій.
2.  Встановити Python 3 (якщо не встановлено).
3.  Виконати скрипт у терміналі:

```bash
python main.py
```
